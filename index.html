Siap main?

Tekan Start Game, lalu klik target neon secepat mungkin.

Cara Main

1. Klik target

Setiap klik benar menambah skor dan combo.

2. Jangan klik kosong

Miss akan mengurangi combo dan skor.

3. Naik level

Target makin kecil saat skor semakin tinggi.

4. Pecahkan rekor

Best score tersimpan otomatis di browser.
